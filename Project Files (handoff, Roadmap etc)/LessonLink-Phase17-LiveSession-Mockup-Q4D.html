Build notes for VS Code Claude

This is a visual reference mockup. Open in a browser to see the intended design. The full build spec with types, schema, file structure, and implementation order is in LessonLink-Phase17-LiveSession-Spec-Q4D.md.

Fonts: Contrail One (display/headings) + Poppins (body). Both loaded via Google Fonts.

Palette: Kiddoland brand colors — navy (#404376), slate (#686ea8), orange (#f2811d), pink (#fe598b), peach (#f8dab9), lavender (#e2d6f4), ice-blue (#dcebf4).

Key principle: This is for kids aged 5-12. Big type, round corners, chunky borders, warm language, celebration everywhere. Not a dashboard — a game screen.

Animations shown here are simplified. The real build needs: twinkling stars (CSS pulse), moving comets (CSS translate), the full Wow screen-flash, Treasure Chest open sequence, and smooth card slide-ins.

The center zone must stay empty — T's webcam sits there via ManyCam.
